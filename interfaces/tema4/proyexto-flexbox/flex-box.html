Wrap

Esta propiedad especifica que los elementos se ajustaran si es necesario

.flex-container {
display: flex;
flex-wrap: wrap;
}

Nowrap

Esta propiedad especifica que los elementos NO se ajustaran

display: flex;
flex-wrap: nowrap;
}

Wrap-reverse

Esta propiedad especifica que los elementos se ajustaran si es necesario, pero en orden inverso
